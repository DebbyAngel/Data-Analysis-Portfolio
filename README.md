### 3. HNG Slack Workspace Exploratory & Predictive Analysis
**Tools:** Python (pandas, NumPy, Scikit-learn, Matplotlib) | Jupyter Notebook | Power BI | Google Docs

### 6. Web Google Graph Network Analysis
**Tools:** Python (NetworkX, pandas, Matplotlib) | Jupyter Notebook | Google Docs
